# Referências de Design para Landing Page de Seguradora

## Análise de Padrões Observados

### Paletas de Cores Comuns
1. **Azul Profissional**: Azul escuro (Navy/Dark Blue) com tons de azul claro - transmite confiança e segurança
2. **Amarelo Vibrante**: Usado como cor de destaque para CTAs e elementos interativos
3. **Verde Suave**: Associado a saúde e proteção
4. **Branco/Cinza Claro**: Backgrounds limpos e profissionais

### Elementos de Design Recorrentes
- **Hero Section**: Imagem de pessoas/família + headline forte + CTA principal
- **Ícones Minimalistas**: Representando categorias de seguros (carro, casa, família, negócio)
- **Cards com Sombras Suaves**: Para apresentar produtos/categorias
- **Tipografia Clara**: Sans-serif moderno (Inter, Poppins, Montserrat)
- **Espaçamento Generoso**: Layouts não-centralizados, com assimetria estratégica

### Estrutura de Navegação
- Menu fixo no topo com logo e links principais
- Seções bem definidas com separadores visuais
- Botão flutuante para ação principal (WhatsApp, Chat, CTA)
- Rodapé com informações de contato e links

### Padrões de Interação
- Hover effects suaves em botões e cards
- Transições fluidas entre seções
- Gradientes sutis para profundidade
- Ícones que reforçam a mensagem visual

## Decisões de Design para Este Projeto

### Paleta Escolhida
- **Primária**: Azul profundo (#1e40af ou similar)
- **Secundária**: Laranja/Coral para destaque (#f97316)
- **Neutra**: Cinza escuro para texto, branco para backgrounds
- **Acentos**: Verde suave para elementos de confiança

### Tipografia
- **Display**: Fonte bold para headlines (ex: Poppins Bold)
- **Body**: Fonte legível para conteúdo (ex: Inter Regular)
- **Hierarchy**: 3 níveis claros (H1, H2, Body)

### Componentes Principais
1. Header/Navigation - Sticky, logo + menu
2. Hero Banner - Imagem + headline + CTA
3. Simulação - Formulário interativo
4. Institucional - Visão, Missão, Valores com ícones
5. Categorias - Grid de produtos com cards
6. Parceiros - Logo carousel
7. APIs - Seção técnica com documentação
8. Footer - Links, contato, redes sociais
9. WhatsApp Button - Flutuante, fixo
